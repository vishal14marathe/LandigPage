# TTM Fleet Management Landing Page

A modern, responsive landing page for TTM (Total Transport Management) built with React.js and Vite.

## Features

- Responsive design that works on all devices
- Modern UI/UX with clean design
- Accessible components
- SEO optimized
- Fast performance with Vite
- Component-based architecture

## Tech Stack

- React.js 18
- Vite
- CSS3 with CSS Variables
- Google Fonts (Inter)

## Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm or yarn
